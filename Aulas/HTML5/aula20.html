Formulários

Utilizando a linguagem de programação PHP, que aprenderemos mais à frente, o formulário pode ser preenchido por um usuário. Exemplos de formulários são as telas de login de determinado site de compras, ou caixas de input que pedem informações ao usuário, como nome, telefone, endereço, e-mail etc. Neste caso, iremos criar um formulário de login do usuário.

Para criar o formulário, utilize a tag (form). A primeira etapa é inserir todos os componentes dentro do formulário. Assim que o usuário preencher e clicar em enviar, tudo o que estiver dentro da tag (form) será enviado. Abaixo, algumas considerações:

  • A tag (input) - Dentro da tag input existem dois atributos. O type, que define o tipo do componente e o segundo é o name, que define uma forma de recuperar os dados que o usuário digitar utilizando a linguagem de programação. Neste caso, no input type, inserimos o tipo "text". Apesar de ter colocado "login" no atributo name, ainda vamos ver como exatamente deveremos recuperar os dados ali inseridos, em outra etapa deste curso.

  • Vale ressaltar também que, ao invés do type ser text, você pode alterar para password, assim, quando o usuário digitar a senha, ela não irá aparecer diretamente na caixa de input.

  • Perceba que, no atributo "button" não foi inserido nenhum conteúdo em name, porque não há nada que ele irá recuperar com o botão, apenas enviar o que estiver dentro da tag (form). Ao invés disso, foi inserido o atributo "value" que define o texto que irá no botão.

  • Como definimos o tipo do botão como button, nada acontece. Porém, para enviar os dados inseridos pelo usuário, o tipo do botão deve ser "submit". Por enquanto, não veremos como fazer o processamento dos dados, pois precisamos de uma linguagem de programação que será vista mais para frente neste curso.

  • Outra informação importante é a de que, quando inserimos o login e a senha e clicamos em enviar, no atributo submit, no endereço da página aparece tanto o login como a senha do usuário. Obviamente não é nem um pouco seguro, por isso, mais à frente no curso, veremos como retirar essas informações diretamente do endereço do site. O atributo name envia os dados para uma próxima tela e, nessa próxima tela, o desenvolvedor consegue fazer o processamento utilizando a linguagem de programação.

  • O atributo "action" na tag (form) define para onde os dados serão enviados. Poderia enviar, por exemplo, para a página processa.php. Essa página capta os dados utilizando o name e, por exemplo, verificar a autenticidade deste usuário, salvar em um banco de dados etc. Em nenhuma das tags de form foi inserido o atriburo action porque ainda não estamos na etapa do curso onde iremos enviar os dados do usuário.
