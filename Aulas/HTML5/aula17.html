Seção 3: HTML5 - Aula 17 - Links

Links

Cada vez que um usuário clica em um link ele é direcionado para uma página da web.

Primeiro Projeto

Na pasta de Primeiro Projeto, nós criamos dois arquivos para navegar de uma página para a outra. Neste caso, desta aula, utilizamos este arquivo aula17 apenas como a aula em si, pois os arquivos html foram criados no folder Primeiro Projeto. Os arquivos criados foram:

  • index.html

  • contato.html

Index é um nome padrão para a definição de uma página principal em um website.

Para criar links, utilize a tag (a) que quer dizer anchor. Confira nas páginas do Primeiro Projeto clicando aqui.

Além disso, você pode inserir links de outros sites diretamente dentro da tag (a), conforme podemos ver ao clicar neste link que será direcionado para o site do Palmeiras, por exemplo. Lembrando que:

  • O atributo title também cabe para link, como podemos ver neste link para o site do Palmeiras.

  • Quando um link redirecionar diretamente para um site, o endereço deve estar completo, partindo do http://...

  • Nos casos de links não preenchidos, o navegador irá apenas recarregar a página, como se estivesse utilizando o comando F5.
