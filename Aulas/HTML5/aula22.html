Caracteres Especiais

Nesta aula, aprendemos um pouco mais sobre caracteres especiais.

Como os sinais de maior e menor são utilizados como um código HTML, o browser não o interpreta. Por exemplo, se digitarmos "(p)", utilizando os sinais de maior e menor entre a letra p, informando ao browser que aquele código HTML é um início de um parágrafo, e não o fecharmos com a barra após inserir o conteúdo, o browser nada irá exibir ali. Por isso, existem as entities html.

Entities HTML

Ao acessarmos este link para o site do w3schools, na área de HTML, temos as entidades. As entidades servem para que o browser interprete determinado código e o transforme naquele caractere especial que o desenvolvedor precisa utilizar. Por exemplo, se ele quiser utilizar o sinal de maior, o código é &+gt+;, ficando ">". A lista contendo os códigos estão neste site ou na tabela abaixo. Portato, agora, o desenvolvedor pode escrever os caracteres, conforme a tabela:

Vale lembrar também que o código &nbsp; cria um espaço em branco, pois sabemos que o navegador não interpreta espaços em branco entre as palavras. Mesmo que o desenvolvedor insira no corpo to texto, vários espaços em branco, o navegador irá considerar apenas 1 espaço em branco, para separar as palavras e letras. Veja no exemplo abaixo:

Hello World (sem o espaço em branco)
Hello       World (com 5 espaços em branco).
