Bordas

No conteúdo desta aula, a <div> é o nosso container. O id #conteudo é onde as bordas foram aplicadas. No id, a propriedade utilizada para as bordas é border. O desenvolvedor então define três valores. São eles:

  • Espessura - A espessura é definida em pixels.
  • Tipo - O tipo da borda define se ela será sólida, pontilhada etc.
  • Cor - Por último, defina a cor da borda.


Todo o conteúdo inserido dentro da propriedade border não contém vírgulas nem ponto e vírgula, assim definindo como será esta borda. Nós podemos inserir a propriedade border em praticamente todos os elementos HTML que estão dentro de <body>. Tags como <p>, <span>, <div>, entre outras, também recebem a propriedade border. Percebe-se, das tags acima, que as tags <span> e <a> são tags inline e as tags <p> e <div> são tags em bloco. Lembrando também que estamos utilizando um id para inserir as bordas, quando o correto seria utilizar as classes.

Lembrando que algumas bordas irão alterar bastante e outras não, sempre conforme a cor inserida. Por isso, a cor é muito importante para a visibilidade da borda. Na borda "hidden" parece que não existe borda nenhuma, porém ela existe. Ela serve para inserir o espaçamento entre os itens. Já a borda "none", essa realmente não existe.

Sobre a borda com duas ou mais propriedades, vale lembrar sempre que ela começa do topo, vai para a lateral direita, a base e, por fim, a lateral esquerda, sempre no sentido horário.
