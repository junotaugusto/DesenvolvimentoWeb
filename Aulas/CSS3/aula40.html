Modelo de Caixa

As tags HTML se comportam com o modelo de caixa, ou seja, é como se cada uma das tags fosse uma caixa. Confira a imagem abaixo:




Lembrando que o padding é um espaçamento interno, ou seja, ele serve para movimentar o que estiver DENTRO da caixa.


Já a margin serve para movimentar o que estiver FORA da caixa.


Nesta aula, uma nova pasta intitulada "Projeto CSS3 Intermediário 2022" foi criada. Dentro dela, um arquivo "modelo_de_caixa.html" serviu como base para a aula. Clique aqui para abrir o arquivo.


Vale lembrar também que, quando temos dois elementos próximos, a margem utilizada será a maior entre os elementos. Por exemplo, temos duas caixas, uma em cima da outra. Na de cima a margem do bottom é de 10px e a margem do top da de baixo é de 20px. A margem que será levada em consideração será a margem da caixa de baixo, onde a margem do top é de 20px. Portanto, a distância entre elas será de 20px.
