A propriedade "Float" é quem define o elemento flutuante. No caso, foram criadas três divs e duas classes diferentes. Uma delas contendo a propriedade "float". Quando utilizamos o "float", seria como se o conteúdo flutuasse acima de outro conteúdo que se encontra "pregado" no "piso" do site. Como se nós olhássemos o site de cima e um dos elementos começasse a flutuar.

Outra coisa importante é que quando um elemento está flutuando ele não se sobrepõe a outro. Portanto, se uma div estiver orientada a flutuar para a esquerda e outra div tiver duas vezes a altura desta, e ela estiver orientada a flutuar para a esquerd, ela não ficará por cima, conforme vemos aqui no site.

Para criar um texto ou uma tag que fique abaixo do conteúdo flutuante, porém que esteja fora do fluxo flutuante e não fique em volta dos elementos flutuamtes, como um texto por exemplo. Para isto, foi utilizado a propriedade "clear". Nesta propriedade, é definido onde o conteúdo será aplicado, por isso ela possui as opções de "left", "right", "both" e "none".

Sem a propriedade "clear" o texto ficava como na imagem abaixo:



Ao aplicar a propriedade "clear: left" nós jogamos os textos para fora do fluxo dos elementos flutuantes, deixando o site como ele se encontra agora.

Outro ponto desta aula é referente ao comportamento de divs dentro de um fluxo flutuante. No exemplo abaixo temos três divs flutuantes dentro de uma div com um id intitulado "principal". Quando aplicamos um background ao id, ele não aparece dentro da div por possuir elementos flutuantes. O conteúdo, além de não aparecer, aparentemente se comporta de forma que ele não aparenta estar ali.

Portanto, se o desenvolvedor quiser, por exemplo, aplicar uma imagem ao background daquela div e, dentro desta div, outras divs serem elementos flutuantes, o background não será aplicado de forma correta.

Por isso, a propriedade clear também deve ser aplicada. O clear limpa o fluxo flutuante e, com isso, automaticamente, este conteúdo vai expandir. Conforme vemos na imagem acima, ao aplicar "clear", como foi feito em uma "div fantasma", o conteúdo da id aparece normalmente.

Lembrando também que os elementos flutuantes não empurram o conteúdo do container como, por exemplo, na div acima, quem empurra o conteúdo da div pai é o elemento que possui a propriedade "clear". Se tivessemos um texto menor do que a altura dos elemmentos flutuantes, o container ficaria do tamanho deste texto.
