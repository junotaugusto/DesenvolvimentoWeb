Divs e Span

Nesta aula, utilizaremos as Divs e Spans para estruturar melhor o site. As divs servem para dividir o site. Inicialmente, no conteúdo desta aula, foi criado uma div para o logo e uma div para os menus..

A ideia é utilizar divs para dividir o site em partes diferentes. A tag para utilizar uma div é <div>.

Conteúdo da aula

Dentro de uma div, nós podemos inserir formatações. Inserimos o logo da amazon abaixo atribuindo a width de 150 pixels.

A <div> possui um id criado para a formatação do topo do site. Nós não conseguimos inserir a formatação de alteração de background (pelo menos eu não sei ainda) aplicada diretamente na Tag da <div>. Por isso, a cor de fundo foi inserida por meio do ID, definido no estilo css.

Mais pra frente no curso, aprenderemos a converter a lista abaixo da imagem em um menu diferenciado.

Basicamente, foram inseridas duas ids no site. Porém, não é o recomendável. A ideia aqui é mostrar como o site se comporta com as divs inseridas.

Tanto a logo como a lista ficaram cinza porque esta é a formatação do id do topo da página. Já as três divs abaixo estão cada uma com uma classe diferente, conforme escrito no código. O fundo vermelho se dá porque estas três divs estão dentro de outra div que está com um id cujo background é vermelho. Além disso, a margem entre as divs é o que evidencia estas diferenças.

A <div> por padrão ocupa todo o espaçamento da tela. Por isso, se o usuário diminuir o tamanho da tela, ela irá acompanhar o tamanho da tela. Nós definimos a div como um elemento do tipo Block. Elementos do tipo Block ocupam todo o espaço da tela. É como se eles tivessem uma largura de 100%.

Já a tag <span> é um elemento do tipo inline. Diferentemente de quando aplicamos um estilo diretamente em alguma tag HTML e chamamos isso de estilo inline, o elemento inline quer dizer que ele agrupa os seus elementos linha a linha.

Por isso, a recomendação para a formatação de textos dentro de uma <div> é sempre utilizando a tag <span> conforme utilizado abaixo no texto "Todos direitos reservados".
