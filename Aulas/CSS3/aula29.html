Seção 4: CSS3 - Aula 29 - Seletores (External Style Sheet) CSS

Folhas de Estilo

Existem três maneiras de trabalhar com CSS. Não é a toa que a nomenclatura é CSS3. São elas:

  1. Inline Style - Neste caso, o desenvolvedor aplica a formatação direto no elemento. (Aula 27)
  2. Internal Style Sheet - Ao criar um estilo utilizando esta maneira, o desenvolvedor aplica o estilo diretamente dentro do código. (Aula 28)
  3. External Style Sheet - Quando o desenvolvedor aplica as formatações do site em um arquivo externo. Este estilo é o mais recomendado e o que mais iremos utilizar neste curso.

External Style Sheet

Primeiro item a ser destacado é o ID. O ID dentro de um site identifica um ponto único do site. Não existem IDs duplicados em nenhum site. Já a Classe, é possível utilizá-la diversas vezes. Utilizando uma classe o desenvolvedor pode definir vários elementos com a mesma classificação.

Para aplicar uma formatação em um parágrafo, por exemplo, pode ser feito como está no site, onde o seletor da tag <p> está configurado para que todo texto dentro de um parágrafo esteja com o alinhamento justificado. Existe esta maneira de fazer isso, mas, se o desenvolveor quisesse que um parágrafo específico tivesse um alinhamento central, ele poderia ir diretamente ao parágrafo em questão e, dentro da tag, inserir o atributo Style="text-align: center;". Outra forma de trabalhar de forma específica é utilizando as Classes.

As Classes servem para que em algum ponto do seu site seja formatado conforme determinada classe pede. Confira no consteúdo abaixo das duas linhas. Elas são definidas com um ponto "." e podem ter o nome que o desenvolvedor escolher. O exemplo utilizado aqui é de uma classe chamada de "destacada".

Já o ID, utilizamos o símbolo de cerquilha, ou hashtag e definimos o nome para identificar aquele elemento. A recomendação é a de utilizar IDs para estruturar o site.
