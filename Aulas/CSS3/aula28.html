Seção 4: CSS3 - Aula 28 - Seletores (Internal Style Sheet) CSS

Folhas de Estilo

Existem três maneiras de trabalhar com CSS. Não é a toa que a nomenclatura é CSS3. São elas:

  1. Inline Style - Neste caso, o desenvolvedor aplica a formatação direto no elemento. (Aula 27)
  2. Internal Style Sheet - Ao criar um estilo utilizando esta maneira, o desenvolvedor aplica o estilo diretamente dentro do código. (Aula 28)
  3. External Style Sheet - Quando o desenvolvedor aplica as formatações do site em um arquivo externo. Este estilo é o mais recomendado e o que mais iremos utilizar neste curso.

External Style Sheet

Seria uma folha de estilo interna, pré-definida diretamente no arquivo do site. Dentro da tag <head> o desenvolvedor pode adicionar Javascript e as folhas de estilo. Portanto, dentro da tag <head> é que o desenvolvedor irá inserir as informações tanto para a formatação quanto para a execução de códigos do site.

A primeira coisa a se fazer, portanto, é abrir a tag <Style>. Depois o desenvolvedor insere os seletores que receberão a formatação, ou seja, as tags que serão alteradas, conforme imagem abaixo:



No exemplo acima, a tag que receberá a formatação é a tag <p>, de parágrafo. No caso, todo o conteúdo de texto ficará com a cor vermelha. Portanto, não é necessário, a partir dali, que o desenvolvedor fique alterando a cor do texto, parágrafo por parágrafo. A tag, em si, recebe a formatação.

Qualquer outra tag pode receber formatação, como a tag <body> por exemplo, fazendo com que, tudo o que esteja dentro do body seja formatado com as alterações impostas na folha de estilo. Entretanto, é preciso reforçar que as alterações obedecerão o que estará inserido dentro de cada tag. Por exemplo, se a tag <body> receber um background azul e a tag <p> receber um background verde, o background do body, mesmo contendo o parágrafo, não sobrepujará a cor dentro da tag de parágrafo, mantendo o que está especificado por cada seletor diferente.

Caso o desenvolvedor abra outro arquivo, a mesma formatação precisará ser repetida no código. Por isso, veremos como trabalhar com External Style Sheet, onde usaremos um arquivo de fora para realizar a formatação de todos os arquivos HTML.

  1. Tag <p>
    1. Alteração na cor do texto para "red".
    2. Alteração na cor de fundo para "lightskyblue".
    3. Alinhamento do texto para justificado.
  2. Tag <h1>
    1. Alteração na cor do texto para "darkgreen"
    2. Alteração na cor de fundo para "ghostwhite
